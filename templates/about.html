Le Procédé d'ExtractionExtraction Process
1

Découverte de Liens (Link Discovery)Link Discovery

Avant de scraper, le système analyse la page d'accueil pour trouver des sous-pages de promotions cachées (ex: /offres, /deals). Il utilise un système de scoring par mots-clés pour prioriser les liens les plus pertinents. Before scraping, the system analyzes the home page to find hidden promo sub-pages (e.g., /deals, /offers). It uses a keyword scoring system to prioritize the most relevant links.

2

Scraping Adaptatif (Scrapling)Adaptive Scraping

Le moteur bascule entre deux modes : HTTP Ultra-rapide pour les sites simples, et Navigateur Réel (Playwright) pour les sites complexes comme Tim Hortons ou DoorDash. Il simule des comportements humains pour contourner les protections anti-bot. The engine toggles between two modes: Ultra-fast HTTP for simple sites, and Real Browser (Playwright) for complex sites like Tim Hortons or DoorDash. It simulates human behavior to bypass anti-bot protections.

3

Analyse par Intelligence ArtificielleAI Analysis

Le code HTML est purgé de tout contenu inutile (scripts, styles) pour ne garder que le texte. Ce texte est envoyé à GPT-4o-mini qui extrait les noms de plats, les prix, et les dates sous un format JSON structuré. The HTML code is purged of all useless content (scripts, styles) to keep only the text. This text is sent to GPT-4o-mini which extracts dish names, prices, and dates in a structured JSON format.
